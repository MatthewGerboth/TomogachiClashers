This Java project is a console-based card battle simulator inspired by strategy games like Clash Royale.
Users can choose a card character, give it a name, and interact with it by performing actions such as eating, leveling up, evolving, attacking, and viewing stats.

Each card has unique attributes and behaviors, but all cards follow a shared structure defined by an interface.
